### 2. Star Label Refinements (User Feedback Round)
User requested:
- **Bigger toggle buttons** — Layout (Pill | Stacked) and Tag Color (Player | Universal) use 13px/600-weight buttons
- **No leash lines** — default false, user reported gray artifacts were unwanted leash lines
- **Color mode toggle** — replaced Owner Border/Fill booleans with `Player | Universal` mode. Universal mode reveals HSLA sliders
- **Border width slider** — 0-5px, set to 0 to hide border
- **Removed inline toggle** — replaced by layout toggle (Pill/Stacked)

### 3. Main Menu Design Concept (Documented, Not Implemented)
User described a 4-column step-based game setup flow:
1. **Choose A Map** — Classic | Custom | New, each with card thumbnails
2. **Choose Opponents** — SP: AI count/settings; MP: lobby browser + create
3. **Mode & Options** — game rules configuration
4. **Start** — large inviting button

**Critical constraint:** Current main menu is NOT to be replaced. New design is a **non-destructive option** togglable via a sticky topnav toggle, allowing user to explore both layouts.

### 4. MP Lobby Concept (Documented)
- Built into main menu, not a separate screen
- Host assigns slots to AI or human
- Lobby stays open until human slots fill
- Host can change map/settings after lobby creation but before game start
- Unified map selection for SP and MP
