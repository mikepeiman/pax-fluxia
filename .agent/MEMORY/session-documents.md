

# Session Documents

## MANDATORY: Save a dated session document every session

At the end of (or during) every session, save a dated session log to:
`.agent/WIP Work-In-Progress/SESSION_YYYY-MM-DD.md`

### Document should include:
- Session focus/title
- Bug fixes completed (with IDs, root cause, fix)
- Features implemented
- Documentation changes
- Roadmap items documented
- Open items deferred
- Key files modified
- Lessons learned (if any)

This ensures no work is lost between sessions.
