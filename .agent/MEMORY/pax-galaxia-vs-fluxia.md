# Pax Galaxia vs Pax Fluxia — Scope Rule

## CRITICAL: Keep These Strictly Separate

### Pax Galaxia (Active Development)
- **Goal**: 100%-fidelity recreation of the 2007 indie game Pax Galaxia
- **Scope**: ALL core gameplay mechanics, visuals, combat, production, repair, orders, AI, star types
- **Status**: Active implementation and bug fixing
- **This is what we are building NOW**

### Pax Fluxia (Roadmap / Future Evolution)
- **Goal**: An evolution of Pax Galaxia with additional features, options, and dynamics
- **Scope**: New mechanics not in the original (star upgrades, hybrid stars, conditional orders, strategic patterns, spectator mode, etc.)
- **Status**: ROADMAP ONLY — NOT for active development
- **These features are tracked in FEATURE_STATUS.md under "Planned Features - Not Started (R)"**

## Rule
- NEVER conflate roadmap/speculative features with core gameplay
- Retreat, scatter, production, repair, star types, AI — these are CORE, not roadmap
- Star upgrades, branching orders, conditional orders, spectator mode — these are ROADMAP (Pax Fluxia)
