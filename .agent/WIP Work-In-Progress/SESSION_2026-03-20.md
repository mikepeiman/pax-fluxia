### 1. Fix Transition Splice Detection (3 bugs)
**Root cause**: `findRingSpliceWindow` did a linear prefix/suffix walk on span arrays, but ring construction puts spans in different order between frames (e.g., prev starts with `ai-4|human-player`, next starts with `human-player|world`). This caused prefix=0/suffix=0 → 0% stationary → full-territory morphs / snap / fold.

**Fixes applied**:
1. `findRingSpliceWindow.ts` — rewrote with circular span matching (try all rotational alignments, pick best)
2. `sampleTransitionFrame.ts` — whole-ring interpolation fallback instead of snap when splice fails
3. `PowerVoronoiRenderer.ts` — borders drawn from splice morph geometry, not separate polyline data
