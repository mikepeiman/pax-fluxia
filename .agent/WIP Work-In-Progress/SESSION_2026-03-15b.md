### 2. Sound Manifest 404 Fix
**Root cause**: `soundManifest.ts` had `import.meta.glob('/static/sounds/**/*')` which Vite interprets as module resolution from the static dir — but public dir files aren't in Vite's module graph. This generated 404 errors in dev.
**Fix**: Removed the entire `discoverSoundFilesViaGlob()` function — it was dead code (never imported/called). The hardcoded `SOUND_MANIFEST` is the actual source of truth.
