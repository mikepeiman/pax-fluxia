### 2. Transition Bug Diagnosis
User reported: fills go black, borders fracture on conquest. Root causes identified:
- **Fills**: chainWalk junction matching fails because OT-interpolated endpoint ptKeys differ from static polyline ptKeys (ptKey rounds to 2 decimals)
- **Borders**: orientation mismatch (prev polyline runs X→Y, next runs Y→X), rotational alignment for closed polygons
- **Core issue**: anonymous geometry with no identity, orientation, or structural correspondence
