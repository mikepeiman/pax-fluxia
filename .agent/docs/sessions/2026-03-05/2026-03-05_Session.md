## Technical Notes

### Shader Architecture (Current State)
```
Pass 1 (ownership): Star data texture → fragment shader loop (0..uNumStars)
  → influence = pixelDist + w * dijkstra (morphed)
  → output: R=owner, G=gapNorm (SDF), B=enemy, A=1

Pass 2 (visual): Reads ownership RT
  → fill: player color + HSL adjustment + uFillAlpha
  → GPU SDF borders: smoothstep on gapNorm (asymmetric — needs fix)
  → junction smoothing: alpha *= smoothstep(0, uSmoothing, gapNorm * 200/smoothing)
  → edge fade: symmetric using uContentMinX/Y
```

### Key Discovery: gapNorm ≠ True SDF
- `gapNorm = clamp(influenceGap / 200.0, 0.0, 1.0)` where `influenceGap = bestEnemyInfluence - bestOwnerInfluence`
- This is an **influence metric**, not a **geometric distance**
- The gradient steepness varies spatially: steep near close stars, shallow near distant ones
- This causes border width to vary with local influence landscape
- A true euclidean SDF to the ownership boundary would require neighbor sampling (Option 2) or a multi-pass jump flood algorithm
