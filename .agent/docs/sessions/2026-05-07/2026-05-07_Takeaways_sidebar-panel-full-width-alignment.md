# Takeaways - 2026-05-07 - Sidebar Panel Full-Width Alignment

## Lessons
- Fixing viewport overflow is not the same as fixing section alignment.
- In this surface, the correct owner is the full action section card.
- If the user says the left edge is wrong, inspect disclosure ancestry first, not just width and overflow values.

## Rule
- For the in-game right-sidebar action surface, any expanded save/load disclosure should default to full section width unless the user explicitly asks for per-button flyouts.
