# FG3 Implicit Trace

## Plain-Language Explanation

`FG3` treats the frontier as a contour inside an implicit field. Instead of assembling frontiers from discrete seeds first, it traces the equal-value boundary between competing ownership fields.

## Current Implementation State

- Status: partial
- Native maturity: not yet native end-to-end
- Current role: selectable comparison path for implicit/field-style territory ideas

## Current Live Routing And Backend Reality

- Live only when `static` mode is active and `FG3` is selected
- Current backend reality: adapter-backed through `DF`
- Dynamic and hybrid routes that use `FG3` do so through their own route definitions, not through the visible static picker

## Target Shape

`FG3` should become the native implicit contour publisher of:

- pairwise implicit fields
- traced frontiers with closure validation
- canonical holdings/components after junction resolution

## Backend Fit

- `DF`: strongest natural fit for field-heavy comparison and parity
- `PVV3`: important target once the implicit contours publish canonical artifacts cleanly
- `PVV2`: maintained comparison backend rather than the conceptual home

## Dependencies

- `EPIC_01_foundation_and_contracts`
- field and screenshot validation from `05_VALIDATION_AND_DEMO_PROTOCOL.md`

## Atomic Tasks

- `FG3-001` Implement pairwise implicit field construction
- `FG3-002` Implement predictor-corrector contour tracing and closure validation
- `FG3-003` Implement junction resolution and canonical publish
- `FG3-004` Validate FG3 on all three backends

## Acceptance Checks

- Contours close correctly under stress
- Junction resolution produces shared frontier geometry rather than duplicated edges
- Holdings/components publish without ownership gaps
- Backend validation exists on `PVV2`, `PVV3`, and `DF`

## Screenshot And Demo Scenarios

- Static `FG3` screenshots on medium and enclave maps
- At least one screenshot proving contour closure around a complex junction region
- One backend comparison screenshot against `FG2`

## Stop Conditions

- Stop if contour tracing looks plausible but does not publish canonical shared edges
- Stop if closure is claimed without saved evidence on a stress map
