# Session notes — 2026-04-09

## Metaball GPU blur: optional border blur

- **Config:** `METABALL_BLUR_AFFECTS_BORDERS` (default `false`) in `game.config.ts` — when `true` and `METABALL_BLUR > 0`, blur applies to a shared PIXI layer (fill + border strokes); otherwise only fill is blurred.
- **Renderer:** `MetaballRenderer.ts` — `metaballLayer` + `ensureMetaballParenting`, fingerprint includes the flag, `applyBlurFilter` / `resetMetaballCache` updated.
- **UI:** Territory controls — checkbox “Blur affects borders” after GPU blur; `PANEL_CONFIG_MAP` entry `metaballBlurAffectsBorders`.
- **Tunables:** `MetaballFamily.ts` `METABALL_TUNABLE_KEYS`; `categoryThemes.ts` territory bundle.
